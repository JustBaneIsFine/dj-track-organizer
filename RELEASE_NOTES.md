## v0.1.4.2

- Fixed a freeze that could happen when picking a folder in Check folder. It now uses the system folder picker and shows progress while matching.
- Unticked matches are now remembered: a file you mark as "not the same song" won't be offered for that track again.
- Opening Check folder from an artist's panel now closes the panel so the results aren't hidden behind it.
- In the revisit view, "open download link" now goes straight to the real download page instead of SoundCloud's redirect.
- Notes search now matches by word, so "Baile, Jersey" finds notes written "Baile,House,Jersey".

100% local: no accounts, no telemetry. Provided as is, use at your own risk. Not affiliated with SoundCloud.
